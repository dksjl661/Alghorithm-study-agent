# Study Notes

## Merge Sort
Merge sort is a divide-and-conquer sorting algorithm.
It recursively splits a list into halves until sublists are size one,
then merges sorted sublists into a final sorted list.
Time complexity is O(n log n) in best, average, and worst cases.

## Binary Search
Binary search works on sorted arrays by repeatedly halving the search interval.
Time complexity is O(log n).

## Recursion
Recursion is a problem-solving technique where a function calls itself with a smaller input.
A correct recursive solution needs a base case and a recursive step.
